# Gerenciamento de Cinema

Projeto final da disciplina de Linguagem de Programação em Python. Este sistema permite o gerenciamento de um cinema, incluindo funcionalidades para clientes e administradores.

## Funcionalidades

### Para Clientes:
- **Login e Registro**: Clientes podem se registrar e fazer login no sistema.
- **Compra de Ingressos**: Opção para comprar ingressos.
- **Visualização de Sessões**: Clientes podem visualizar as sessões disponíveis.

### Para Administradores:
- **Gestão de Clientes**: Visualizar, adicionar e excluir clientes cadastrados.
- **Gestão de Filmes**: Visualizar, adicionar e excluir filmes cadastrados.
- **Gestão de Sessões**: (Funcionalidade a ser implementada).

## Tecnologias Utilizadas
- **Python**: Linguagem principal do projeto.
- **CustomTkinter**: Biblioteca para criação de interfaces gráficas.
- **MySQL**: Banco de dados para armazenamento das informações.
